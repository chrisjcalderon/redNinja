Welcome to RedNinja Library - Easy, Fast and mean!.

A easy to use library for developing MVVM applications built on top of popular Open Source Frameworks.

By Christian Calderon Contact me @ LinkedIn

The framework uses at it's core the following amazing Open Source libraries

  • CUJOS CURL: https://github.com/cujojs/curl
  • JQUERY: http://jquery.com/
  • KnockOut JS: http://knockoutjs.com

Optional, but included because they are awesome:

  • XDate JS: http://arshaw.com/xdate/
  • DataTables: http://datatables.net/
  • Jquery UI: jqueryui.com
  • KO External Template Engine: Github site...
There are two types of template models: one based on bootstrap and the other Gantry-framework. They each have a model that allows you to register sections/positions and modules dynamically - they should help a lot in organizing your page content and separate everything into small modules

<strong>Yes, curl.js follows the CommonJS Loader Plugin specification, 
<br>so you can use any compatible plugin. The following plugins are included:</strong>
<ul>
    <li>js! -- loads non-AMD javascript files. more info on the wiki</li>
    <li>text! -- loads text files</li>
    <li>link! -- loads css files via a link element (simple, fast)</li>
    <li>css! -- loads css files (lots of options)</li>
    <li>domReady! -- resolves when the dom is ready for manipulation</li>
    <li>async! -- resolves when a module signals it's ready</li>
</ul>
</div>
